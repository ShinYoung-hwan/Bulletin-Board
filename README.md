# Bulletin Board

# 다운로드

```bash
> git clone https://github.com/ShinYoung-hwan/Bulletin-Board.git
> cd board
> yarn install
```

# 사용법

- 서버 실행

```bash
> node server.js
```

- 서버 접속 [http://localhost:3000/]

# 추가 업데이트 목표

- database json에서 MySQL로 변경

# 사용 기술

- Frontend: express-handlebars, bootstrap
- backend: express
- database: json
